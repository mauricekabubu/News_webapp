# Hot News - Flask News Website

## Overview
**Hot News** is a dynamic news website built using Python, Flask, HTML, CSS, and Bootstrap**. It fetches real-time news articles from the [News API](https://newsapi.org/) and displays them in a clean and user-friendly interface.  

The goal of this project is to practice **backend development, API integration, and web deployment**, while creating a functional project that can be used for portfolios, internships, and freelance work.

---

## Features

- Fetches **latest news articles** from News API
- **Search functionality** to find news by keyword
- **Category filters** (e.g., Technology, Sports, Health)
- **Responsive design** using Bootstrap for desktop and mobile
- **Real-time analytics** tracking via Google Analytics
- Clean and readable **user interface** with intuitive navigation
- Deployed online and accessible via: [Hot News Website](http://mauricekabubs.pythonanywhere.com/)

---

## Tech Stack

- **Backend:** Python, Flask  
- **Frontend:** HTML, CSS, Bootstrap  
- **APIs:** News API for fetching news  
- **Analytics:** Google Analytics (GA4)
